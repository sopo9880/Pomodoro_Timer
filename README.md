# Bloomodoro

감각적인 UI와 설치형 PWA 지원이 포함된 포모도로 타이머입니다.

## 실행

정적 파일만으로도 동작하지만, 설치형 앱과 모바일 테스트를 위해서는 로컬 서버를 권장합니다.

```bash
node server.js
```

이후 브라우저에서 `http://localhost:4173`으로 접속하면 됩니다.

## 기능

- 포모도로, 짧은 휴식, 긴 휴식 모드
- 예쁜 진행 링과 반응형 모바일 UI
- 세션 자동 전환, 기록 저장, 알림음
- 브라우저 시스템 알림, 다음 세션 안내 토글, 현재 사이클 전체 초기화
- 실제 경과 시간을 반영하는 집중 시간 누적 계산
- 워커 기반 백그라운드 타이머와 실행 중이던 타이머 복구 로직
- 지원 브라우저용 미니 타이머 창
- PWA 설치 지원
- 오프라인 캐시 지원

## 알림과 백그라운드 동작

- PC에서는 시스템 알림 권한을 허용하면 다른 작업 중에도 세션 완료를 운영체제 알림으로 받을 수 있습니다.
- 백그라운드에서는 메인 스레드 대신 워커가 타이머를 유지해 긴 세션에서 드리프트를 줄입니다.
- 크롬 계열 데스크톱 브라우저에서는 미니 타이머 창을 띄워 작은 상시 표시 형태로 사용할 수 있습니다.
- 웹 앱은 휴대폰 무음 모드를 강제로 해제하거나, 화면이 꺼진 상태에서 네이티브 알람 앱처럼 시스템을 항상 깨우는 것은 지원할 수 없습니다.
- 대신 앱을 다시 열면 저장된 종료 시각을 기준으로 누락된 세션 진행 상황을 복구합니다.
